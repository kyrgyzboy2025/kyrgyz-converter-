# Қырғызча⇆قىرعىزچا

在线将柯尔克孜语在阿拉伯字母和西里尔字母之间双向转换。

## 🌐 立即使用
访问：https://kyrgyzboy2025.github.io/KyrgyzConverter/

## ✨ 功能特色
- 🔄 双向转换：阿拉伯字母 ↔ 西里尔字母
- 📱 响应式设计：手机、平板、电脑都能用
- 📋 一键复制结果
- 📜 本地历史记录（自动保存）
- ⚡ 快速转换：Ctrl+Enter快捷键
- 🎨 现代化UI界面

## 📖 使用说明
1. 在文本框中输入柯尔克孜语文本
2. 点击"转换"按钮或按Ctrl+Enter
3. 使用"交换方向"按钮切换转换方向
4. 点击"复制结果"将转换文本复制到剪贴板
